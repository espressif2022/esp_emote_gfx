scene.json 字段速查

顶层：schema name prefix meta.viewbox layout parts poses clips sequence

poses[*].parts 存各部件控制点； clips[*].steps 为设备端离散 step（非时间轴）。

流程：打开 / 拖入 JSON → 识别并整理（可选）→ 编译并仿真 → 导出 .inc。
