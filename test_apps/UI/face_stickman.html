Stickman Static Action Board

这个页面直接以 fireman.svg 的火柴人比例为基础，只保留圆头和直线四肢的读形。 每个动作都是单个静态目标姿态，不自动循环；只有在你点击动作按钮时，才会从当前姿态缓动切到下一个姿态。 现在走路和跑步已经拆成左右各 step1 / step2 两组，导出时会带上分组和步序信息，方便你在设备端循环切换。

Shape Rule

头保持圆形，身体、手臂、双腿都只用直线。走、跑、跳、挥手、坐下只通过肢体角度和端点变化来表达，不额外加表情或装饰。

Transition Rule

页面默认停在当前动作，不会自己播。点击另一个动作后，用和设备侧一致的阻尼参数把当前姿态推向目标姿态。
