Stickman Static Action Board

这个页面直接以 fireman.svg 的火柴人比例为基础，只保留圆头和直线四肢的读形。 画面改成黑底白人，方便先看剪影是否成立；每个动作仍然都是单个静态目标姿态，不自动循环， 只有在你点击动作按钮时，才会从当前姿态缓动切到下一个姿态。现在预览区额外带了 Board / Device 两种模式： Board 适合逐个看姿态，Device 会按设备端同样的方向触摸映射来拖拽切动作。走路和跑步继续拆成左右各 step1 / step2，导出时会带上分组和步序信息，方便你在设备端循环切换。 Board 下可点 + 从平静姿态新建自定义动作，拖动关节点后 Save 保存（数据在浏览器本地存储）。

Preview Mode
上体/下体分段联动；肘、膝始终落在肩-手、髋-脚连线上。自定义动作可点 ✎ 命名、× 删除。

Shape Rule

头保持圆形，身体、手臂、双腿都只用直线。网页里统一用黑底白线先看剪影，动作只靠关节点位置变化来成立，不额外加表情或装饰。

Transition Rule

Board 模式下仍然是点按钮看静态姿态；Device 模式下可以直接在预览区拖拽，上下左右和斜向会映射到设备侧同款动作。
